== Description ==

Embed or create a podcast on your WordPress blog with our plugin. Just use our shortcodes or user-friendly button and put the link to your .mp3 file and that's it.

Add Podcast Player to your post with the [podcast] [/podcast] shortcodes:

Example:

[podcast title="Podcast Name"]http://www.example.com/audio.mp3[/podcast]

Credits:

audio.js - http://kolber.github.io/audiojs/

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
